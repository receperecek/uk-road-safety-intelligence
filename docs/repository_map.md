# Repository Map

## data/
Contains raw, reference, processed and analytical outputs.

## docs/
Contains project charter, source definitions and design notes.

## src/
Will contain reusable Python pipeline scripts.

## notebooks/
Reserved for controlled exploratory work, not primary pipeline logic.

## outputs/
Contains charts and final analytical tables for reporting.

## reports/
Contains final markdown summaries and project writeups.

## config/
Contains scope and project-level configuration files.
