# UK Road Safety Intelligence Platform

## Business Case
This project analyzes official UK Department for Transport STATS19 road safety data
to identify high-risk patterns, severity drivers, vulnerable road user exposure,
and intervention priorities.

## Why this project matters
Road safety analysis should go beyond counting collisions.
The goal is to support better decision-making by understanding:
- where risk concentrates
- which conditions increase severity
- which user groups are more vulnerable
- where intervention priorities should focus

## Project Scope (v1)
- Time window: 2020-2024
- Geography: Great Britain
- Data model: collisions + vehicles + casualties
- Focus: data cleaning, relational modeling, KPI design, severity analysis, hotspot analysis

## Out of Scope (v1)
- 2025 provisional data
- Sensitive contributory factor data
- Full historical dataset (1979-2024)
- Cloud deployment
